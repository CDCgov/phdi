Public Health Data Infrastructure (PHDI) API Documentation
====================================================================

Welcome to the PHDI API documentation!

This project is aimed at providing public health departments with tools to improve data ingestion, facilitate interjurisdictional collaboration, and streamline reporting.

For information on how to get started using or contributing to PHDI Building Blocks, see our `GitHub Repository <https://github.com/CDCGov/phdi>`_.
