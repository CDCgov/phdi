phdi.linkage.link module¶

phdi.linkage.link.generate_hash_str(linking_identifier, salt_str)[source]¶

Generates a hash for a given string of concatenated patient information. The hash serves as a "unique" identifier for the patient.

Parameters

  • linking_identifier (str) -- The value to be hashed. For example, the concatenation of a patient's name, address, and date of birth, delimited by dashes.

  • salt_str (str) -- The salt to use with the hash. This is intended to prevent reverse reverse engineering of the PII used to create the hash.

Returns

The hash of the linking_identifier string.

Return type

str
