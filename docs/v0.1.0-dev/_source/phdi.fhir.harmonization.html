phdi.fhir.harmonization.standardization module¶

phdi.fhir.harmonization.standardization.standardize_names(data, trim=True, case='upper', remove_numbers=True, overwrite=True)[source]¶

Standardize all names contained in a given FHIR bundle or a FHIR resource. The default standardization behavior is our defined non-numeric, space-trimming, full capitalization standardization, but other modes may be specified.

Parameters

  • data (dict) -- A FHIR-formatted JSON dict.

  • trim (bool) -- Whether leading/trailing whitespace should be removed. Default: True

  • case (Literal['upper', 'lower', 'title']) -- The type of casing that should be used. Default: upper

  • remove_numbers (bool) -- If true, delete numeric characters; if false leave numbers in place. Default: True

  • overwrite (bool) -- If true, data is modified in-place; if false, a copy of data modified and returned. Default: True

Returns

The bundle or resource with names appropriately standardized.

Return type

dict
