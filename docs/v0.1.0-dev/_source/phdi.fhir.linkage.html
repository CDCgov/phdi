phdi.fhir.linkage.link module¶

phdi.fhir.linkage.link.add_patient_identifier(patient_resource, salt_str, overwrite=True)[source]¶

Given a FHIR Patient resource:

  • Extract name, DOB, and address information

  • Compute a unique hash string based on these fields

  • Add the hash string to resource

Parameters

  • patient_resource (dict) -- The FHIR patient resource to add a linking identifier.

  • salt_str (str) -- The salt to use with the hash. This is intended to prevent reverse reverse engineering of the PII used to create the hash.

  • overwrite (bool) -- Whether to write the new standardizations directly into the given bundle, changing the original data. Default: True

Returns

The resource updated with additional patient identifier.
