phdi.fhir.download_from_export_response(export_response)[source]¶

Accepts the export response content as specified here: https://hl7.org/fhir/uv/bulkdata/export/index.html#response---complete-status

Loops through the "output" array and yields the resource_type (e.g. Patient) along with TextIO wrapping ndjson content.

Parameters

export_response (dict) -- JSON-type dictionary holding the response from the export URL the FHIR server set up.

Return type

Iterator[Tuple[str, TextIO]]

phdi.fhir.export_from_fhir_server(cred_manager, fhir_url, export_scope='', since='', resource_type='', container='', poll_step=30, poll_timeout=300)[source]¶

Initiate a FHIR $export operation, and poll until it completes. If the export operation is in progress at the end of poll_timeout, use the default polling behavior to do the last function check before shutting down the request.

Parameters

  • cred_manager (phdi.azure.AzureFhirServerCredentialManager) -- Service used to get an access token used to make a request.

  • fhir_url (str) -- FHIR Server base URL

  • export_scope (str) -- Either Patient or Group/[id] as specified in the FHIR spec (https://hl7.org/fhir/uv/bulkdata/export/index.html#bulk-data-kick-off-request)

  • since (str) -- A FHIR instant (https://build.fhir.org/datatypes.html#instant) instructing the export to include only resources created or modified after the specified instant.

  • resource_type (str) -- A comma-delimited list of resource types to include.

  • container (str) -- The name of the container used to store exported files.

  • poll_step (float) -- the number of seconds to wait between poll requests, waiting for export files to be generated.

  • poll_timeout (float) -- the maximum number of seconds to wait for export files to be generated.

Return type

dict

phdi.fhir.export_from_fhir_server_poll(poll_url, cred_manager, poll_step=30, poll_timeout=300)[source]¶

The main polling function that determines export file availability after an export run has been initiated.

Parameters

  • poll_url (str) -- URL to poll for export information

  • cred_manager (phdi.azure.AzureFhirServerCredentialManager) -- Service used to get an access token used to make a request.

  • poll_step (float) -- the number of seconds to wait between poll requests, waiting for export files to be generated. defaults to 30

  • poll_timeout (float) -- the maximum number of seconds to wait for export files to be generated. defaults to 300

Raises

  • polling.TimeoutException -- If the FHIR server continually returns a 202 status indicating in progress until the timeout is reached.

  • requests.HTTPError -- If an unexpected status code is returned.

Return type

requests.models.Response
