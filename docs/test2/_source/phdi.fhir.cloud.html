phdi.fhir.cloud.azure module¶

phdi.fhir.cloud.azure.download_from_fhir_export_response(export_response, cred_manager)[source]¶

Accept the export response content as specified here: https://hl7.org/fhir/uv/bulkdata/export/index.html#response---complete-status

Loops through the "output" array and yields the resource_type (e.g. Patient) along with TextIO wrapping ndjson content.

Parameters

  • export_response (dict) -- JSON-type dictionary holding the response from the export URL the FHIR server set up

  • cred_manager (phdi.cloud.azure.AzureCredentialManager) -- Service used to get an access token used to make a request

Returns

An iterator of tuples. Each tuple is comprised of: - FHIR resource type (str) - Export file content (io.TextIO)

Return type

Iterator[Tuple[str, TextIO]]
